## Windows向けビルド方法について

VisualStudioが必要です。（直近では、Visual Studio 2019で確認）

 x64 Native Tools Command Prompt

を開き、ソースコードのあるディレクトリにて、

nmake -f Makefile.vs

もしくは

nmake -f MakefileAVX.vs

を実行すると、ソースコードと同じディレクトリに、obj/exeファイルが
（ビルドに成功すれば）出来ます。

(警告がいっぱい出るのですが、致命的なものはないので、無視して下さい。)
